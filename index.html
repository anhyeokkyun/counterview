⏱️ COUNTERVIEW README

Firebase Realtime Database 기반 양방향 동기화 웹 타이머

✨ 주요 기능 (Key Features)

  • 실시간 타이머 동기화: 호스트의 제어 상태(시작, 일시정지, 초기화, 잔여 시간)가 뷰어 화면에 무지연 실시간 동동기화됩니다.
  • 중간 알림 기능 (Split): 목표 종료 시간 외에 특정 중간 도달 알림 시간을 세팅하여 비프음 사운드와 텍스트 블링크 경고를 줄 수 있습니다.
  • 자체 웹 오디오 시스템: 무거운 오디오 파일 없이 Web Audio API 브라우저 내장 오실레이터를 조합해 비프음 사운드 알림을 출력합니다.
  • 뷰어 전용 템플릿: 공유 링크(viewer=1)로 접근 시 복잡한 설정창이 전부 가려지고 초대형 타이머 디스플레이만 노출됩니다.
  • 모바일 반응형 구축: 기기 노치 스페이스(Safe Area) 방어 및 가로 회전 모드 대응 레이아웃이 적용되었습니다.

🛠️ 기술 스택 (Tech Stack)

  • Frontend: HTML5, CSS3 (Flexbox, clamp() 반응형 폰트), JavaScript Modules
  • Backend / DB: Firebase Realtime Database
  • API / Built-in: QR Code Generator API, Web Audio API
